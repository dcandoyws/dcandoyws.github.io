hexo d 权限问题

Posted on 2019-08-06 | In Git

hexo github “Permission denied”

当你弄完一篇文章想把它上传到你的github上时，突然发现没有权限了–“Permission denied ”，这是什么原因？

其实大概率原因就是你的github的SSH key过期了，是不是过期呢，检查一下，在hexo根目录git bush，输入ssh-keygen”命令，它会提示你的.ssh文件在哪里，你只需要找到id_rsa.pub,找不到的话，那么就是你的SSH key过期了，现在怎么办呢？别急，就继续用刚刚这个命令不过要加点东西，ssh-keygen -t rsa -C "邮箱名",然后就继续回车就好了，出现类似以下内容：

说明你已经创建好了，现在找到这个文件（id_rsa.pub）复制其中内容到你的github新加的sshkeys里，大功告成，接下来就可以hexo d到你的github了
